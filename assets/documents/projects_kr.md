공정·제조 데이터, 운영형 ML, AI-ready data를 실제 의사결정과 업무 도구로 연결한 프로젝트입니다. 내부 테이블명, 상세 시스템명, 법인명, 금액 효과는 제외하고 공개 가능한 범위로 정리했습니다.

## Deep Case Studies

### 사용자 단말 센서 품질 로그(DQA) Data Agent / AI-ready Data 구축

품질 raw log를 Agent가 해석 가능한 mart와 knowledge로 재구성해, 하드웨어 개발자가 자연어로 품질 로그를 분석할 수 있게 만든 프로젝트입니다.

- **메타:** 2026-현재 · 사용자 단말 품질 로그 · AI-ready data · 정식 운영 및 확대 진행
- **문제:** 스마트폰 부품·센서 품질 로그가 JSON 문자열 형태로 대량 수집되지만, key별 의미·단위·정상 범위·전처리 방식이 달라 SQL과 하드웨어 지식 없이는 분석이 어려웠습니다.
- **내 역할:** spec 문서와 실제 인입 데이터의 gap을 비교하고, key별 의미·단위·전처리 규칙·값의 범위·분석 시 주의점을 정리했습니다.
- **접근:** raw log 직접 조회를 줄이기 위해 mart table을 설계하고, LLM용 table/column description, domain knowledge, Question-SQL pair, 검증용 기준 query를 작성했습니다.
- **결과:** 사용자는 모델별 error log 발생률, 국가별 온도 구간 비중, 전체 디바이스 대비 특정 조건 발생 비율 등을 자연어로 질의하고 SQL 결과와 해석을 받을 수 있게 되었습니다.
- **역할 경계:** SQL 생성·검증·실행·해석 파이프라인은 별도 개발 조직이 담당했고, 저는 data/knowledge preparation과 하드웨어 담당자와의 key 의미 검증을 맡았습니다.

### TV 공정 단계 잠재 불량 예측

공정 직후 정상으로 보이는 TV 세트 중 이후 시장불량으로 이어질 가능성이 높은 제품을 조기에 선별한 프로젝트입니다.

- **메타:** 2018.11-2020.10 · 제조/공정 품질 · 지도학습 · 현업 적용
- **문제:** CS 단계에서 불량을 대응하기보다 공정 직후 잠재 불량 단계에서 조기에 잡아내는 것이 비용과 브랜드 측면에서 유리했습니다.
- **내 역할:** 공정 검계측 데이터와 전세계 서비스센터의 CS 불량·수리 이력을 serial number 기준으로 연결하고, 이후 불량 발생 가능성을 예측하는 학습 문제로 구성했습니다.
- **접근:** 데이터 매핑·전처리, 의미 단위 학습 데이터 구성, 초기 모델링 코드, 초기 평가 metric 코드를 직접 구현했습니다.
- **운영:** positive로 예측된 세트는 aging test를 우선 진행해야 했기 때문에, 라인 병목과 탐지 효과 사이의 균형을 고려해 threshold를 현업과 협의했습니다.
- **결과:** 희소 불량과 시기별 baseline 차이를 고려해 lift 지표를 제안했고, 주요 feature와 유사 과거 데이터 경향을 함께 전달했습니다. 적용 기간 중 관련 불량률은 절반 수준으로 낮아졌습니다.

### BLU Simulator 개발 및 검증

miniLED TV의 백라이트 구조를 실물 제작 전에 시뮬레이션하고, 화질 기준을 만족하는 광학 옵션을 탐색한 프로젝트입니다.

- **메타:** 2020-2021 · 광학/패널 개발 · 시뮬레이션 · Windows 프로그램 배포
- **문제:** 신규 백라이트 방식에서는 광원 간격, 광원-패널 거리, 구동 조건에 따라 uniformity, 블루밍, 블록단차가 달라졌고, 모든 조합을 실물 패널로 제작해 평가하기 어려웠습니다.
- **내 역할:** 70여 가지 단일 광원 광분포를 parametric curve로 fitting하고, 광원-패널 간격 변화에 따른 parameter 변화를 회귀분석으로 추정했습니다.
- **접근:** 인접 광원의 광분포가 덧셈으로 중첩되는지 검증해 전체 패널 백라이트 시뮬레이션 로직을 만들고, uniformity와 소벨 필터 기반 블록단차 지표를 설계했습니다.
- **결과물:** Python으로 구현한 뒤 PyQt와 packaging 도구로 Windows 프로그램을 배포했습니다. 사용자는 광원 분포, 간격 범위, 구동 제약조건을 입력하고 설계 옵션 후보를 확인할 수 있었습니다.
- **결과:** 실물 패널 촬영 이미지와 시뮬레이션 밝기 map의 평균 유사도 90% 이상을 확보했고, miniLED 개발 프로세스 적용, 개발 기간 절반 수준 단축, 재료비 70% 이상 감축으로 이어졌습니다.

### Trade-in Fraud Detection

Trade-in incentive를 악용하는 의심 주문을 구매 시점에 탐지하고, 주문 담당자가 검토할 수 있게 한 운영형 ML 프로젝트입니다.

- **메타:** 2025.07-2026.03 · eCommerce risk · XGBoost 분류 모델 · 운영 이관
- **문제:** 기존 기기 제출 약속만으로 즉시 할인을 제공하는 구조에서, 일부 주문이 기기를 제출하지 않고 회수 절차도 회피해 손실로 이어졌습니다.
- **내 역할:** EDA, feature 발굴, 모델 학습, threshold 설계, dashboard 제공 정보 정의 등 데이터와 모델링 전반을 수행했습니다.
- **접근:** 주문, 고객, 구매 방식, 제품 수령 방식, 적용 할인, 제출 예정 구형 모델, 과거 구매 이력, 회수 절차 로그를 활용해 fraud risk feature를 만들었습니다.
- **운영:** probability distribution 변화에 따라 threshold를 조정해 탐지 물량을 제어하고, positive 주문은 주요 feature, 전체 feature 값, 과거 주문 정보와 함께 담당자 dashboard로 전달했습니다.
- **결과:** 일부 주문 적용군과 미적용군의 사후 결과를 활용하는 A/B test 유사 방식으로 검증했고, 운영 기준 precision 70% 수준을 확인했습니다.

## Additional Case Studies

### Enterprise Knowledge System / Ontology / 지식 그래프 PoC

상용 LLM이 모르는 사내 용어, 시스템 사용법, 데이터 의미, 부서별 convention을 구조화해 AI가 활용할 수 있게 만드는 PoC입니다.

- **메타:** 2026-현재 · 사내 지식 시스템 · Ontology/RDF/지식 그래프 · PoC/아키텍처 정의
- **문제:** 사용자가 매번 긴 사내 맥락을 prompt에 넣는 방식으로는 확장성이 낮고, 사내 암묵지와 데이터 활용 지식은 상용 LLM이 기본적으로 알 수 없었습니다.
- **내 역할:** Ontology 설계 PoC, RDF 변환 규칙, entity/relation 추출 기준, 기본 지식 그래프 구축, 상충·모호·불일치 지식 탐지 PoC를 수행했습니다.
- **결과:** 샘플 지식에 대해 Ontology와 지식 그래프를 만들고, 상충 definition, 오래된 문서, 모호한 질의 mapping, 데이터값-지식 규칙 불일치 탐지 예시를 구현했습니다.
- **역할 경계:** LLM Wiki schema 자체 설계와 전사 운영 시스템 배포가 아니라, 지식 관리 방식과 PoC 구현 단계의 프로젝트입니다.

### 서비스 자재 / TV 패널 수요예측

단종 이후 서비스 교체에 필요한 TV 패널 수요를 예측해 과잉 재고와 재고 부족 비용을 함께 줄이려 한 프로젝트입니다.

- **메타:** 2018.10-2020 · 장기 수요예측 · 유사자재 매핑 · 시스템화 T/F
- **문제:** TV 패널은 모델과 연식에 따라 호환이 어려워 단종 시점에 서비스용 패널을 미리 확보해야 했고, 적게 확보해도 많이 확보해도 비용이 발생했습니다.
- **내 역할:** 문제 정의, 데이터 전처리, 유사자재 매핑 로직 구현, 전체 수량과 월별 trend를 분리해 예측 정확도를 판단하는 아이디어 제안에 깊게 관여했습니다.
- **접근:** 유사자재를 스펙이 아니라 판매·서비스 수요 패턴으로 정의하고, 판매 trend와 서비스 trend의 형태 유사성을 활용했습니다.
- **결과:** benchmark 오차율 29.6% 대비 19.1% 수준까지 개선되어 목표 오차율을 달성했고, 현업 시스템 적용과 발명 출원으로 이어졌습니다.
